# InfraSense AI 🏗️🤖
**Predicting Infrastructure Risks Before They Happen.**

### 📌 Overview
InfraSense AI is an AI-powered monitoring engine developed for the **Mini Smart India Hackathon 2026** (Problem Statement: MSIH25030). It addresses cost overruns and timeline delays in major infrastructure projects.

### 🚀 Key Features
* **Predictive Risk Engine:** Forecasts probability of delays and overruns.
* **What-if Simulator:** Interactive dashboard to test project variables (Budget, Timeline, Contractor Quality).
* **XAI Lite:** Provides factor attribution to show *why* a project is at risk.

### 🛠️ Tech Stack
* **Frontend:** React.js (Vite), Tailwind CSS, Recharts.
* **Backend:** Python (Flask), REST API.
* **Machine Learning:** Scikit-Learn (Random Forest Regressor).
